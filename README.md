# MA-Cross-Trading-Bot

A Python-based trading bot for backtesting and simulating trades using historical stock data. This bot calculates Simple Moving Averages (SMA) and Long-term Moving Averages (LMA) to generate buy/sell signals and outputs trade reports in CSV and PDF formats.

---

## Features

- **Historical Data Analysis:** Downloads stock price data from Yahoo Finance.
- **Technical Indicators:** Calculates SMA and LMA for trading signals.
- **Trading Strategy:** Generates buy/sell decisions based on moving average crossovers.
- **Backtesting:** Simulates trades and calculates final balance.
- **Report Generation:** Outputs trading results in CSV, PDF, or other user-friendly formats.
- **Easy to Extend:** Modify strategy, timeframe, or tickers with minimal changes.

---

Stock ticker, LMA length, and SMA length can be edited in the code.

The bot has a backtester built in and will output a table with the results.
